見出し

見出しは
<h1>見出しの内容</h1>
で記述します。

見出しは

  1. 大きいh1

  2. 二番目h2

  3. 三番目h3

のように番号で重要さが振り分けられている。

段落

段落は <p>文章</p>
と記述します 　

改行は
</br> と記述する。

各パーツの名前

  • タグ

    • <>で囲まれた部分

  • 属性

    • タグに追加的な情報を付けることができる。

  • 属性値

    • class = "ranking"
    classが属性で、rankingは属性値です。
    属性値は、属性を識別するための名前です。
