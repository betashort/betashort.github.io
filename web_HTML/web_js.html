javascriptの導入

javascpritを導入するには、方法が２つあります。 １つ目は、HTMLファイルに直接書プログラムを書き込む方法です。 ２つ目は、プログラムを書いたjsファイルを用意して、それをHTMLファイルにて読み込む方法です。

直接書き込む場合

headの中に<script>プログラム</script>
のように書き込みます。

一見、便利で簡単そうに思えますが、プログラムが巨大になる程、管理しづらくなると思います。 なので、次のjsファイルを読み込む方をおすすめします。

jsファイルを読み込む方法

こちらの方法も、同様にheadの中に記述します。
<script type="text/javascript" src="jsファイル名"></script>
と記述します。CSSを導入するときも別ファイルを用意しましたが、javascriptも同様に管理した方が良いでしょう。 僕は、こちらを採用します。
