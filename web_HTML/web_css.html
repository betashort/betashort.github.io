CSS

CSSは飾り付けをする。

CSSは、透過処理・グラデーション・角丸・ドロップシャドウ・アニメーションなどの表現できる。

ブラウザのバージョンによって、対応が十分でない場合がある。

書き方

セレクタ{プロパティ:値;}

コメント

HTML <!--コメント-->

CSS /* コメント */

CSSの外付け

HTMLのhead内に記述する

<link rel="stylesheet" href="css.css">

CSSファイルの最初は
@charset "UTF-8";
と記述する。
